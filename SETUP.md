# LinkWords ‚Äî App Store Setup Guide

## 1. Firebase Project

1. Go to https://console.firebase.google.com
2. Create new project ‚Üí "LinkWords"
3. Enable **Authentication** ‚Üí Sign-in methods:
   - Google (enable)
   - Apple (enable ‚Äî requires Apple Developer account)
   - Facebook (enable ‚Äî requires Facebook Developer app)
4. Enable **Cloud Firestore** ‚Üí Start in production mode
5. Deploy `firestore.rules` from this repo
6. Go to Project Settings ‚Üí General ‚Üí Your apps ‚Üí Add Web App
7. Copy the Firebase config values into `auth.js` ‚Üí `FIREBASE_CONFIG`

## 2. Apple Developer Setup (for Apple Sign-In)

1. Go to https://developer.apple.com
2. Certificates, Identifiers & Profiles ‚Üí Identifiers
3. Register App ID: `com.linkwords.app`
4. Enable "Sign in with Apple" capability
5. Create a Service ID for web sign-in (for PWA)
6. In Firebase Console ‚Üí Authentication ‚Üí Apple ‚Üí add the Service ID

## 3. Facebook Developer Setup (for Facebook Login)

1. Go to https://developers.facebook.com
2. Create new app ‚Üí Consumer type
3. Add Facebook Login product
4. In Firebase Console ‚Üí Authentication ‚Üí Facebook ‚Üí add App ID and App Secret from Facebook

## 4. Capacitor ‚Äî iOS

```bash
npx cap add ios
npx cap sync
npx cap open ios
```

In Xcode:
- Set Team & Bundle ID (`com.linkwords.app`)
- Add "Sign in with Apple" capability
- Add `GoogleService-Info.plist` from Firebase Console
- Build & run on simulator or device

## 5. Capacitor ‚Äî Android

```bash
npx cap add android
npx cap sync
npx cap open android
```

In Android Studio:
- Add `google-services.json` from Firebase Console ‚Üí `android/app/`
- Set signing config for release
- Build & run

## 6. App Store Submission

### iOS (App Store Connect)
- Screenshots: 6.7" (iPhone 15 Pro Max), 6.5" (iPhone 11 Pro Max), 12.9" iPad Pro
- App category: Games ‚Üí Word
- Privacy URL: https://linkwords.app/privacy
- Age rating: 4+

### Android (Google Play Console)
- Screenshots: phone + 7" tablet + 10" tablet
- Category: Games ‚Üí Word
- Content rating questionnaire
- Data safety form (collects: email, name via auth; stores: game progress)

## 7. Required Pages (host at linkwords.app)

- `https://linkwords.app/privacy` ‚Äî Privacy Policy page
- `https://linkwords.app/terms` ‚Äî Terms of Service page

Both are required by Apple and Google for app store submission.

## 8. AdMob & In-App Purchases

### AdMob
1. Go to https://admob.google.com ‚Üí Create app (iOS + Android)
2. Create ad units: **Banner**, **Interstitial**, **Rewarded Video**
3. Copy the ad unit IDs into `ads.js` ‚Üí `AD_CONFIG.admob`
4. Copy the app IDs into `capacitor.config.json` ‚Üí `plugins.AdMob.appId`

### In-App Purchase (Remove Ads)
1. **iOS**: App Store Connect ‚Üí In-App Purchases ‚Üí Create non-consumable `com.linkwords.removeads`
2. **Android**: Google Play Console ‚Üí Monetize ‚Üí Products ‚Üí Create `com.linkwords.removeads`
3. The `@capgo/capacitor-purchases` plugin handles both platforms
4. **Recommended price**: $2.99 USD (Tier 3) ‚Äî industry standard for casual word games

### AdSense (Web PWA fallback)
1. Go to https://adsense.google.com ‚Üí Create ad unit
2. Copy client ID and slot into `ads.js` ‚Üí `AD_CONFIG.adsense`
3. For auto-ads, add the AdSense script tag to `index.html` `<head>`

## 9. Environment Files

### iOS: `ios/App/App/GoogleService-Info.plist`
Download from Firebase Console ‚Üí Project Settings ‚Üí iOS app

### Android: `android/app/google-services.json`
Download from Firebase Console ‚Üí Project Settings ‚Üí Android app
